A tree in which we have N nodes and N-1 edges and all nodes are reachable from each other.
There can be multiple spanning trees for a graph, but a spanning tree where the sum of all the edge weights is minimal is called an MST.

Algorithm's to find MST
1. Prim's Algorithm 
2. Kruskal's Algorithm
